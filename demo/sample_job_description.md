# Job Description — Senior Backend Java Engineer

**Title:** Senior Backend Java Engineer  
**Seniority:** Senior (6+ years)  
**Location:** Remote (LatAm) / Hybrid  
**Type:** Full-time

## About the role

We are hiring a Senior Backend Java Engineer to design and evolve microservices that power our B2B platform. You will own critical services end-to-end, improve reliability, and mentor mid-level engineers.

## Required skills (hard requirements)

- Java 17+
- Spring Boot
- Microservices architecture
- AWS (ECS/EKS, SQS/SNS, or equivalent)
- Apache Kafka (or strong event-driven experience)
- SQL (PostgreSQL or MySQL)
- CI/CD and automated testing

## Preferred skills

- Leadership / tech lead experience
- Observability (OpenTelemetry, Prometheus, Grafana)
- Domain-Driven Design
- Mentoring and hiring support

## Experience

- 6+ years backend engineering
- 3+ years building production microservices
- Experience in high-throughput or event-driven systems

## Responsibilities

- Design and implement scalable Spring Boot services
- Own Kafka-based event pipelines
- Improve service reliability and performance
- Collaborate with Product and Platform teams
- Mentor engineers and contribute to technical standards

## Nice to have

- Kubernetes
- Terraform
- Experience in fintech or high-compliance environments
